JavaScript

JavaScript er det mest populære sprog inden for programmering. JavaScript bruges, meget kort forklaret, til at gøre elementerne på din hjemmeside dynamiske ved hjælp af DOM (Document Object Model). Som eksempel, på min hjemmeside bruger jeg metoden "document.getElementById(button1);" for at finde min knap så jeg kan ændre farven på den, når en tekstboks er åben.
