CSS

CSS er et endnu et sprog inden for kodning. CSS er sproget der står bag størstedelen af alt stiliseringen af hjemmesiden. CSS er mit personlige yndlings sprog at kode i, fordi jeg nyder at finde et design som jeg synes ser godt ud.
CSS står for:

C - Cascading
S - Style
S - Sheets

CSS kan ændre på udseendet af elementer i HTML, så længe det er linket. Det har jeg gjort ved at skrive <link rel="stylesheet" type="text/css" href="style.css"></link> i min HTML fil.
